# Module 00 — Python & ML Basics
This module builds the complete foundational toolkit — Python, NumPy, Pandas, PyTorch, TensorFlow, and classical ML — needed before tackling any bioinformatics topic in this curriculum.

## What You'll Learn
- Core Python programming patterns used throughout bioinformatics (data structures, file I/O, comprehensions)
- NumPy array manipulation, Pandas dataframes, and scikit-learn pipelines for ML workflows
- Supervised and unsupervised ML fundamentals: regression, classification, clustering, cross-validation
- PyTorch tensor operations, autograd, and basic neural network training loops
- TensorFlow/Keras model building and training
- Mathematical foundations: linear algebra, calculus for backprop, probability, and statistics
- HackerRank Python and Statistics/ML track problem-solving strategies
- Advanced classical ML: ensembles, SVMs, dimensionality reduction, hyperparameter tuning

## Key Concepts Introduced
- **Autograd**: PyTorch's automatic differentiation engine that computes gradients via dynamic computation graphs.
- **Cross-validation**: Model evaluation strategy that estimates generalization by rotating train/test splits across data folds.
- **Regularization**: Techniques (L1, L2, dropout) that constrain model complexity to reduce overfitting.
- **Dimensionality reduction**: Methods (PCA, UMAP) that compress high-dimensional feature spaces while preserving structure.
- **Ensemble methods**: Algorithms (random forests, gradient boosting) that combine weak learners into a stronger predictor.
